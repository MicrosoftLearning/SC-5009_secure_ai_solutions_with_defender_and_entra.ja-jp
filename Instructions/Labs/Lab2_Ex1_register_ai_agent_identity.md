# ラボ 2 - 演習 1 - AI エージェントのアプリケーション ID を登録する

組織は、Azure AI サービスと Key Vault にプログラムでアクセスする必要がある AI エージェントを構築しています。 資格情報を使用してサインインする対話型ユーザーとは異なり、このエージェントには安全に認証するために独自のアプリケーション ID が必要です。 ID 管理者は、アプリケーションを Microsoft Entra ID に登録し、その認証資格情報を構成します。

**タスク**:

1. アプリケーションを Microsoft Entra ID に登録する
1. 認証を構成し、サービス プリンシパルを識別する

## タスク 1 - アプリケーションを Microsoft Entra ID に登録する

Azure サービスに個別にアクセスする AI エージェントには、Microsoft Entra ID に登録された ID が必要です。 このタスクでは、エージェントを表すアプリ登録を作成し、認証と承認に使用される ID を確立します。

1. **Microsoft Edge** を開き、`https://entra.microsoft.com/` に移動します。

1. ラボ ホスト プロバイダーによって提供されるユーザー名とパスワードを使用してサインインします。

1. 左側のサイド バーの **[Entra ID]** で、**[アプリの登録]** を選択します。

1. **[+ 新規登録]** を選択します。

1. **[アプリケーションの登録]** ページで、次のように入力します。

   - **名前**: `Contoso-AI-Agent`
   - **サポートされているアカウントの種類**: 既定の **[シングル テナントのみ]** オプションを選択したままにします。

1. **リダイレクト URI** は空白のままにします。

    > **注**: このエージェントは、対話型サインインではなくクライアント資格情報を使用して認証するため、リダイレクト URI は必要ありません。 リダイレクト URI は、ユーザーがブラウザー経由でサインインする必要があるアプリケーションに使用されます。

1. **登録** を選択します。

AI エージェントのアプリケーション ID が Microsoft Entra ID に正常に登録されました。

## タスク 2 – 認証を構成し、サービス プリンシパルを識別する

アプリの登録によってアプリケーションが定義されますが、エージェントの ID を証明するには資格情報が必要です。 このタスクでは、認証用のクライアント シークレットを作成し、対応するサービス プリンシパル (割り当てられたロールとアクセス許可を取得する ID オブジェクト) を見つけます。

1. **Contoso-AI-Agent** アプリ登録ページの左側のサイド バーの **[管理]** で **[証明書とシークレット]** を選択します。

1. **[クライアント シークレット]** タブを選択してから、**[+ 新しいクライアント シークレット]** を選択します。

1. **[クライアント シークレットの追加]** ポップアップで次のように入力します。

   - **説明**: `AI agent authentication`
   - **有効期限**: **180 日 (6 か月)**

1. **[追加]** を選択します。

1. 新しいシークレットの **[値]** をコピーします。

    > **重要**: このシークレットの値が表示されるのは 1 回のみです。 運用環境では、この値を手動でコピーするのではなく、Azure Key Vault に安全に格納します。

1. 左側のパネルの [Entra ID] で、**[エンタープライズ アプリケーション]** を選択します。

1. 検索フィールドに「`Contoso-AI-Agent`」と入力し、アプリケーションを選択します。

1. **Contoso-AI-Agent** エンタープライズ アプリケーション ページで、アプリケーションが一覧表示されていることを確認します。

    > **注**: アプリの登録によって、アプリケーションの内容が定義されます。 エンタープライズ アプリケーションはサービス プリンシパルであり、割り当てられたロールとアクセス許可を取得するテナント内の ID オブジェクトです。 Azure リソースへのアクセス権を付与する場合は、このサービス プリンシパルにロールを割り当てます。

認証資格情報が正常に構成され、AI エージェントのサービス プリンシパルが識別されました。
